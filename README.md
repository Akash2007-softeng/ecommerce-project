# ecommerce-project
#  Simple E-commerce Website (Week 1 Project)
##  Project Overview
This is a basic e-commerce website developed as part of my internship project.  
The main objective of this project is to understand the fundamentals of frontend development using HTML, CSS, and JavaScript.
##  Features
- Display of products (Shirt, Shoes, Watch)
- Add to Cart functionality
- Dynamic cart update using JavaScript
- Simple and clean user interface
##  Technologies Used
- HTML
- CSS
- JavaScript
##  Project Structure
ecommerce-project/
│── index.html
│── style.css
│── script.js
│── README.md
## ▶️ How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. The project will open in your web browser
## 📸 Output
- Products are displayed on the screen
- Clicking "Add to Cart" will add items to the cart
- Cart updates instantly below the products
## 🎯 Learning Outcomes
- Learned how HTML, CSS, and JavaScript work together
- Understood basic DOM manipulation
- Gained experience in creating a simple interactive web page
