#  Simple E-commerce Website (Week 1 Project)
##  Project Overview
This is a basic e-commerce website developed as part of my internship project.  
The main objective of this project is to understand the fundamentals of frontend development using HTML, CSS, and JavaScript.
##  Features
- Display of products (Shirt, Shoes, Watch)
- Add to Cart functionality
- Dynamic cart update using JavaScript
- Simple and clean user interface
##  Technologies Used
- HTML
- CSS
- JavaScript
